Fira Sans System Font Replacement
---------------------------------

These fonts are intended as a system font replacement on Mac OS X 10.10 Yosemite.

They are based on the Fira Sans font family and are licensed under the Open Font License version 1.1 or later. This package has been prepared by Jens Kutilek <https://github.com/jenskutilek/FiraSystemFontReplacement>.

How to install:
---------------

Copy the 4 font files into the /Library/Fonts folder. You may have to log out and in again in order to see the change take effect.

How to uninstall:
-----------------

Delete or move the font files from the /Library/Fonts folder.
